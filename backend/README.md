## Prerequisites

- docker >=4.20.0

## Available scripts

### Install and Run

```sh
docker compose up
```

### Run app locally

Needs to have postgreSQL and Redis server running either in docker or locally. Set environment variables in .env file.

Starts api development server at http://localhost:5000/

```sh
yarn start
```
